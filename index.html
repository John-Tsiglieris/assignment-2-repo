How to Apply CSS to HTML

Often times, CSS is applied via a external stylesheet. To create one, create a file named styles.CSS, and write your CSS code in there Make sure it's in the root directory, and add this code to your head tag:

<link rel="stylesheet" href="styles.css?v=1.0">

How to Use JavaScript in a Webpage

JavaScript is used to make dynamic changes to your website. You can do things like changing elements of your HTMl file, and much more.

When your code gets long it's good to import your JavaScript as a separate file. To do this, write your code in a file with a .js file format.

Inside a 'script' tag, link your HTML file to the destination of your JS code. <script src="path/to/my/script.js"></script>
